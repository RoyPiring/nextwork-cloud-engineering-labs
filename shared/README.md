# Shared Components

This directory contains reusable components, patterns, and utilities used across multiple labs.

## Structure

- `infrastructure/`: Reusable Infrastructure as Code modules
- `scripts/`: Utility scripts for common tasks
- `configs/`: Shared configuration templates
- `patterns/`: Code patterns and examples

## Usage

When creating a new lab, check here first for reusable components before creating new ones.

## Contributing

When adding shared components:
1. Ensure they are truly reusable across multiple labs
2. Document usage and examples
3. Include tests
4. Update this README
