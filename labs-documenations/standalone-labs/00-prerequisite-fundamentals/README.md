# Prerequisite Fundamentals

This folder contains foundational labs that are prerequisites for all other end-to-end series. These labs establish the base knowledge and account setup required before beginning any cloud engineering work.

## Labs

### 1. AWS Account Setup
**File:** `aws-account-setup.md`

- Setting up an AWS account
- Understanding AWS Free Tier
- Account configuration and billing setup
- Initial validation and access verification

**Prerequisites:** None - Start here!

### 2. AWS Beginners Challenge
**File:** `aws-beginners-challenge.md`

- Introduction to cloud computing concepts
- Overview of AWS services
- Challenge framework and accountability structure
- Foundation for subsequent lab work

**Prerequisites:** AWS Account Setup

## Purpose

These labs are standalone prerequisites that must be completed before starting any end-to-end series. They provide:
- Account access and billing understanding
- Cloud computing fundamentals
- Learning framework establishment

## Next Steps

After completing these prerequisites, proceed to any of the end-to-end series:
- **standalone-labs/storage** for basic S3 operations
- **series-02-vpc-networking-architecture** for VPC fundamentals
- **series-01-three-tier-application-architecture** for serverless and container workloads
