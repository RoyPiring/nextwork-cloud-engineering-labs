Individual AI/ML tools and techniques that can be used independently or integrated into other projects. These labs are not part of a sequential series and can be completed in any order based on your needs.

## Lab Goals

By completing these labs, you will:
- Understand various AI/ML tools available on AWS
- Implement speech-to-text solutions
- Build and deploy AI agents
- Optimize prompts for better model performance
- Evaluate and compare different AI models

- AWS Account Setup (standalone-labs/prerequisite-fundamentals)
- Basic understanding of AI/ML concepts
- Specific prerequisites noted in each lab
