# Cloud Security & Identity Series

**Rank:** 5 (Feature Project Priority)

This series covers security fundamentals and advanced security services on AWS, implementing defense-in-depth strategies. Demonstrates security-first thinking and governance practices.

## Learning Path

### 1. Identity and Access Management (IAM)
**File:** `aws-security-iam.md`

- IAM fundamentals
- Users, groups, and roles
- Policy creation and management
- Least privilege principles
- Cross-account access

### 2. Key Management Service (KMS)
**File:** `aws-security-kms.md`

- Encryption key management
- Customer-managed keys (CMK)
- Key rotation policies
- Encryption at rest and in transit
- Key access control

### 3. Secrets Manager
**File:** `aws-security-secretsmanager.md`

- Secret storage and rotation
- Integration with applications
- Automatic rotation setup
- Secret access policies
- Cost optimization

### 4. Security Monitoring
**File:** `aws-security-monitoring.md`

- CloudWatch security metrics
- CloudTrail log analysis
- Security event detection
- Alert configuration
- Incident response basics

### 5. GuardDuty
**File:** `aws-security-guardduty.md`

- Threat detection service
- Finding types and severity
- GuardDuty configuration
- Integration with security workflows
- Cost considerations

## Series Goals

By completing this series, you will:
- Implement least privilege access controls
- Encrypt data at rest and in transit
- Manage secrets securely
- Monitor and detect security threats
- Respond to security incidents

## Prerequisites

- AWS Account Setup (standalone-labs/prerequisite-fundamentals)
- IAM basics recommended
- Understanding of security fundamentals
